# File Name Structor Pro

Quick deploy steps:
1. Copy `.env.local.example` to `.env.local` and fill keys.
2. `npm install`
3. `npm run dev` (local) or push to GitHub and deploy with Vercel.

Notes:
- Alias `@/` is enabled via `jsconfig.json`.
- `next.config.js` uses only `reactStrictMode` to avoid experimental warnings.
